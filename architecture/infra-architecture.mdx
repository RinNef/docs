# Kiến trúc hạ tầng

## Stack đề xuất

- **Frontend**: Next.js + React + Tailwind

- **Backend**: Node.js (NestJS / Next.js API routes)

- **Database**: PostgreSQL / MySQL

- **Cache**: Redis (lưu session, token, search cache)

- **Storage**: Object storage (Cloudflare R2 / S3) – lưu ảnh homestay

- **Message Queue (optional)**: RabbitMQ / Kafka – xử lý async (email, webhook)

- **Deploy**:

  - FE: Cloudflare Pages / Vercel

  - BE: Cloudflare Workers / Docker trên VPS
