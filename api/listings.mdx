# Listings API

Các API để quản lý danh sách homestay.

## Business Logic

- Chỉ chủ nhà mới có quyền tạo hoặc chỉnh sửa listing  

- FE chỉ hiển thị các listing có trạng thái `published`  

- Admin có quyền block/unblock  

- Ảnh được upload lên Cloudflare R2 hoặc S3  

## Endpoints

- `GET /listings`

- `GET /listings/{id}`

- `POST /listings`

- `PATCH /listings/{id}`

- `DELETE /listings/{id}`

Mintlify tự fill phần chi tiết từ OpenAPI.
